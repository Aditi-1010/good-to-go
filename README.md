# Node.js Multi-Page Website

This is a basic Node.js server-based website created using core Node modules.

## Features
- Custom HTTP server
- Manual routing
- Multiple pages (Home, About, Contact, Services)
- 404 error handling

## How to Run
1. Install Node.js
2. Clone the repository
3. Run:
   node Aditi62.js
4. Open:
   http://127.0.0.1:3000

## Technologies Used
- HTML
- CSS
- Node.js (http, fs)
